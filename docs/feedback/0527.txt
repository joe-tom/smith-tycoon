1. 용사 전투의 종료 시점이 d+2 내의 언젠가에서 끝나기.
1-1. 전투 결과는 볼 수 없고, 만약 해당 용사가 재방문 하면 결과를 알게 됨.
1-2. 생존시 용사 의견에 따라 2일 후에 재등장 / 부상시 용사 의견에 따라 7일 후에 재등장
1-2-1. 용사 의견이란 : 전투 후 더 좋은 무기를 소유하고 싶거나, 무기가 부서졌다는 결과를 저장하여 제작
1-3. 만약 죽어버리면 d+2 이내의 기간에 우편을 받고 패배하고 죽었음을 알게 됨.

2. 사용자의 하루 루프는 비결정적
2-1. 평판에 따라 방문자의 수가 바뀜 
ex. 평판 0-10 : 3명(고정) 방문 / 11-20 : 3-5명(랜덤) 방문 / 21-40 : 5-7 / 41-60 : 8-10 / 60+ : 10명 고정
2-2. 하루 길이는 제작 단계 직전에 평판을 바탕으로 결정 (하루 중 변동되는 평판은 당일 일정에 영향을 주지 않음)

3. 아이템
3-1. 상인이 day 에 따라 점점 좋은 재료를 가지고 오도록 구현

4. 용사의 재방문
4-1. 살아 돌아온 용사는 그 전리품을 가져와서 대장장이에게 판매하고자 함 -> 거래(협상)
4-1-1. 몹당 제공하는 전리품 종류 + drop 확률 필요
4-1-2. 전리품 대장장이가 사면 호감도 증가

5. chitchat 분기
5-1. 거래가 아닌 호감도에 따른 일정 수준 이상의 대화 가능
5-1-1. chitchat이 끝나는 트리거는 내가 무기를 제시 하는 것. 그리고 제시하면 바로 협상단계로 이동
5-2. 대화 시 용사 정보를 알 수 있고, 저장함.
5-3. 한문단 정도, 짧게

6. 협상 단계
6-1. 인내심 스탯을 영웅, 상인에게 추가
6-2. 인내심 - LLM이 만든 페르소나에 따라 유동적으로 계산
6-3. 인내심이 떨어지면 대화 포기 or 구매 포기 or 터무니없이 저렴한 가격 제시 등 (구현 시 고민)

7. 게임 목표 -> 중간중간 해결해 나가야 하는 작은 미션들 
7-1. 만약 목표 실패가 발생하면 game over
ex1. 열흘마다 세금 징수하러 오는 징수관한테 1000골드를 세금으로 줘야함. -> 3일차에 무조건 세금 징수관 등장 이벤트 -> 협상 x -> 세금 언제 내라고 욕만하다가 감
ex2. 한자동맹 상인조합장이 텃세 부리러 옴 -> 이 NPC 등장하고 d+3 내에 현재 평판 50 도달 -> 11일~15일차 사이에 랜덤하게 등장 -> 상인조합장 퀘스트가 달성되는 즉시 NPC가 재등장 해서 '고생했다' 말하고 감.

8. 무기 칭호
ex. 휴지검 -> 좀비 2번 잡음 -> 좀비 똥을 닦아주는 | 휴지검
