# 대장장이 Tycoon - Architecture

> 프로토타입 설계 문서. 게임 규칙과 데이터 구조를 AI 구현 에이전트가 바로 참조할 수 있도록 정리.

---

## 1. 게임 개요

- **장르**: 대장간 운영 타이쿤 + LLM 기반 NPC 시뮬레이션
- **플레이어 역할**: 대장장이 (User)
- **목표 기간**: 최대 100일
- **핵심 루프**: `제작 → 용사 방문(협상→전투) → 평판/금화 변동 → 다음 날`

---

## 2. 플레이어 (대장장이)

### 2.1 스탯

| 스탯 | 초기값 | 변동 트리거 |
|---|---|---|
| 제작력 | 0 | 무기 제작마다 소량 ↑, 고희귀도 무기 제작 시 추가 ↑ |
| 보유 금액 | 5000 골드 | 용사 판매 시 ↑ / 상인 구매 시 ↓ |
| 평판 | 0 | 거래·전투 결과에 따라 변동 (자세한 규칙은 §6, §7) |

### 2.2 보관 공간

- **인벤토리**: 재료 보관
- **진열장**: 완성품(미판매 무기) 보관

---

## 3. 무기 (Weapon)

### 3.1 제작 공식

```
재료(여러 개) + 제작력 → 무기 인스턴스
무기 이름: LLM이 재료 구성과 재료 스탯을 보고 생성
무기 스킬: LLM이 생성 (단, 용사 구매 확정 전까지 비공개)
```

### 3.2 무기 스탯

| 필드 | 값 |
|---|---|
| 종류 | 한손검 / 양손검 / 한손둔기 / 양손둔기 / 마법지팡이 / 방패 / 단도 / 표창 / 우산 / 양산 / 총 |
| 희귀도 | 0–100 (정규분포). 아래 §3.3 티어 적용 |
| 예리도 | 0–100 (정규분포) |
| 속성 | 물 / 불 / 바람 / 흙 / 금 (5행 상성 적용) |
| 스킬 | LLM 생성. **구매 확정 시 공개** |
| 요구 근력 | 정수 |
| 요구 마력 | 정수 |

### 3.3 희귀도 티어

| 희귀도 | 티어 명칭 |
|---|---|
| 0 – 10 | 습작 |
| 11 – 30 | 망작 |
| 31 – 50 | 평작 |
| 51 – 70 | 수작 |
| 71 – 90 | 명작 |
| 91 – 99 | 전설검 |
| 100 | 신을 죽이는 검 |

- 티어는 UI 표시, 가격 보정, NPC 대사 톤(LLM 프롬프트) 결정에 사용
- `100` 단일 등급은 게임 내 극히 드물게 등장하는 최상위 티어

### 3.4 5행 속성 상성

- 금 → 나무(바람) → 흙 → 물 → 불 → 금 (상극 사이클)
- 전투 데미지 계산 시 보정값으로 사용

## 4. 재료 (Material)

상인이 판매. 총 220종.

| 카테고리 | 종 수 | 등장 확률 | 가격대 | 예시 |
|---|---|---|---|---|
| 일반 | 100 | 높음 | 저렴 | 나뭇가지, 나무판자, 원목, 철덩이, 구리, 고무, 플라스틱 |
| 특수 | 40 | 낮음 | 비쌈 | 다이아몬드, 사파이어, 금괴, 운석 |
| 이상한 | 60 | 중간 | 매우 저렴 | 생선뼈, 계란껍질, 담배꽁초 |
| 전설 | 20 | 극히 낮음 | 매우 비쌈 | 드래곤의 깃털, 테세우스의 배 조각, 아르테미스의 사과, 프로메테우스의 불 |

## 5. NPC

### 5.1 용사

| 스탯 | 설명 |
|---|---|
| 근력 | 무기 요구 근력 충족 여부 영향 |
| 마력 | 무기 요구 마력 충족 여부 영향 |
| 보유 금액 | 첫 방문 시 랜덤 지급, 이후 구매/전리품으로 변동 |
| 기분 | 협상 난이도에 영향 |
| 직업 | 검사 / 법사 / 거렁뱅이 / 궁수 / 성문 문지기 / 청소년 / 군인 등 — 무기 종류별 적합도 수치 보유 |

### 5.2 상인

- 하루 1명 방문
- 재료(전체 풀에서 확률 추출) + 무기(예리도 30, 희귀도 30 고정)를 판매
- **재료는 플레이어가 상인에게 되팔 수 없음**

## 6. 일일 플로우 (Phase)

```
[제작]
  → [용사1 방문 → 협상] → [용사1 전투/결과]
  → [상인 방문 → 협상]
  → [용사2 방문 → 협상] → [용사2 전투/결과]
  → [제작 시간]
  → [용사3 방문 → 협상] → [용사3 전투/결과]
  → [하루 요약/통계]
  → (다음 날)
```

- 하루: 용사 3명 + 상인 1명
- 제작 시간 phase에서만 인벤토리 재료로 무기 제작
- 용사가 무기를 보유한 채 재방문하면, 협상 phase에서 **새 무기 구매** 또는 **무기 강화** 중 택1 (§11)

---

## 7. 협상 시스템

모든 거래는 협상 페이지를 거침. 아무것도 안 살 수도 있음.

### 7.1 용사 → 무기 구매

```
플레이어가 가격 제시
  → 용사 LLM 응답 (수락 / 거절 / 협상)
  → 협상이면 대화 라운드 진행
  → 최종 수락 or 최종 거절
```

| 결과 | 평판 | 기타 |
|---|---|---|
| 구매 수락 | ↑ | 무기 스킬 공개, 보유 금액 ↑ |
| 구매 거절 | ↓ | — |

### 7.2 플레이어 → 상인 재료/무기 구매

```
플레이어가 아이템 복수 선택 → 상인이 total 가격 제시
  → 플레이어가 수락 / 거절 / 협상
  → 협상이면 대화 라운드 진행
  → 최종 수락 or 최종 거절
```

| 결과 | 평판 |
|---|---|
| 구매 수락 | ↑ |
| **협상 후** 거절 | ↓ (즉시 거절은 예외, 평판 변화 없음) |

---

## 8. 마왕군 & 전투

### 8.1 마왕군 병정

- 종류 예시: 작은 영혼, 지옥개, 고블린, 임프, 서큐버스, 케르베로스, 가고일, 데몬 등
- 각 종류마다 **본인 속성** + **약점 속성** + **공격 속성** 데이터 보유
- 난이도 1–100. 같은 종이어도 ±랜덤 (예: 고블린 1–10)

### 8.2 시간에 따른 난이도 곡선

| 일차 | 등장 |
|---|---|
| 1일~ | 저난이도 잡몹 (작은 영혼, 지옥개, 고블린, 임프) |
| 30일~ | 서큐버스 등 중급 |
| 40일~ | **중간보스** 등장 시작 |
| 60일~ | 중간보스 + 데몬/가고일 |
| 80일~ | 중간보스 + **최종보스**만 출현 |
| 100일 | 게임 종료 |

**중간보스 등장 트리거**: 80일 이전이라도 중간보스 7명이 전부 사망하면 최종보스 등장.

### 8.3 보스

- **중간보스 (7명, 7대 죄악)**: 루시퍼(교만), 맘몬(탐욕), 아스모데우스(색욕), 사탄(분노), 벨페고르(나태), 레비아탄(질투), 벨제붑(폭식)
  - 사망 시 재등장하지 않음
- **최종보스**: 수르트 (화염거인 왕)
  - 사망 시 게임 승리

### 8.4 전투 처리

- 용사가 무기를 사지 않았어도 전투 발생 (무기 = null)
- 마왕군 풀에서 랜덤 매칭
- **이미지 없음**. LLM이 뉴스 형태 스크립트로 결과 생성
- 입력: 용사 스탯 + 무기 스탯 + 마왕군 스탯
- 출력: 용사 결과, 무기 결과, 마왕군 결과

### 8.5 전투 결과 → 평판 변화

| 항목 | 결과 | 평판 |
|---|---|---|
| 용사 | 생존 | ↑ |
| 용사 | 부상 | 변화 없음 |
| 용사 | 사망 | ↓ |
| 무기 | 보존 | 변화 없음 |
| 무기 | 파괴 | 소폭 ↓ |
| 마왕군 | 생존 | 변화 없음 |
| 마왕군 | 도망 | 소폭 ↑ |
| 마왕군 | 사망 | ↑ |

### 8.6 전리품

- 용사 생존 시: 전리품 + 골드 획득
- 전리품은 **용사의 다음 방문 시 제작 재료**로 사용 가능 (§10 참조)

---

## 9. 용사 귀환 규칙

| 전투 결과 | 다음 방문 |
|---|---|
| 생존 | 3일 이내 재방문 |
| 도망 | 7일 이내 재방문 |
| 사망 | 더이상 등장하지 않음 |

---

## 10. 용사 전리품 제작 Phase

- 용사가 전투 후 전리품 + 소액 골드를 제공하면 발동
- 전리품 + 인벤토리 재료를 합쳐 무기 생성
- 제작 규칙은 §3과 동일

---

## 11. 무기 강화 (NEW)

> 용사의 무기가 전투에서 **보존**되어 돌아왔을 때, 새 무기 구매 대신 **강화 의뢰**를 받을 수 있음.

### 11.1 발동 조건

- 용사가 재방문할 때 보유 무기가 있으면(직전 전투에서 무기 보존), 협상 phase에서 **새 무기 구매** 또는 **강화 의뢰** 중 하나를 선택
- 두 옵션은 배타적 (한 방문에 둘 다 못함)

### 11.2 강화 입력

```
용사 보유 무기 (1개, 필수)
+ 대장장이 인벤토리 재료 (1~3개, 플레이어 선택)
+ 강화 비용 골드 (용사가 지불, 협상 시스템 재사용)
```

### 11.3 강화 결과

- **예리도 +Δ**, **희귀도 +Δ** (각 0–100 cap)
- Δ는 투입한 재료에 따라 결정되며 **항상 성공** (실패 없음, 상승 폭만 랜덤)

| 사용 재료 카테고리 | 예리도 상승 폭 | 희귀도 상승 폭 |
|---|---|---|
| 일반 | +1 ~ +3 | +0 ~ +2 |
| 이상한 | +0 ~ +2 | +0 ~ +2 |
| 특수 | +3 ~ +7 | +2 ~ +5 |
| 전설 | +7 ~ +15 | +5 ~ +12 |

- 여러 재료 사용 시 각 재료 효과를 합산
- 종류/속성/스킬/요구 근력/요구 마력은 **변경되지 않음**
- 강화 횟수는 무기에 누적 기록 (`enhancement_level`)

### 11.4 협상

- 강화 비용은 §7.1 협상 시스템과 동일하게 처리
- 수락/거절에 따른 평판 변화도 §7.1과 동일
- 호감도 변동도 §12.2와 동일하게 적용 (`바가지` 판정 시 시세는 투입 재료 가격 합 기준)

### 11.5 다른 시스템과의 상호작용

- **시그니처 패턴 카운트에는 포함되지 않음** (시그니처는 신규 제작만 카운트)
- 전설 무기로 등재된 무기를 강화해도 전설 지위 유지
- 강화된 무기가 보스 처치 등 조건을 충족하면 새로 전설로 등재 가능

## 12. 단골 용사 시스템 (NEW)

> 같은 흐름의 반복을 줄이고, "내 대장간 이야기"가 쌓이도록 하는 핵심 장치.

### 12.1 용사 기억 데이터

생존한 용사는 다음을 영구 보유하고 재방문 시 LLM 프롬프트에 주입:

| 필드 | 설명 |
|---|---|
| 이름 | 첫 등장 시 LLM 생성 |
| 성격 태그 | 예: `호탕`, `깐깐`, `소심`, `허세`, `검소` — 최초 등장 시 결정 |
| 말투 샘플 | 일관성 유지를 위해 첫 등장의 대사 일부 저장 |
| 호감도 | -100 ~ +100. 기본 0 |
| 거래 이력 | 최근 5건의 (무기명, 합의가, 전투결과) 요약 |
| 별명 | 큰 활약 시 LLM이 부여 ("○○ 슬레이어" 등) |

### 12.2 호감도 변동 규칙

| 트리거 | 호감도 |
|---|---|
| 적정가 거래 (시세 ±10%) | +5 |
| 후한 거래 (시세 -10% 미만) | +10 |
| 바가지 (시세 +20% 초과) | -10 |
| 구매한 무기로 생존 | +10 |
| 구매한 무기로 사망(귀환 없음) | — (사망 시 시스템에서 제거) |
| 무기 파괴 | -5 |

### 12.3 호감도 효과

| 호감도 구간 | 효과 |
|---|---|
| ≥ +50 | 시작가 +10% 수락, 협상 라운드 단축, 다른 용사를 데려올 확률 ↑ |
| +20 ~ +49 | "지난번 그 검 잘 쓰고 있어요" 같은 회상 대사 가능 |
| -20 ~ -49 | 협상 시 깐깐, 시작가 -10% 요구 |
| ≤ -50 | 구매 거부 가능성 ↑, 부정적 입소문(평판 -1/일) |

### 12.4 회상 대사

재방문 시 LLM 프롬프트에 거래 이력 + 별명 + 호감도를 주입해 자연스러운 회상 대화 생성.

## 13. 시그니처 & 전설 무기 시스템 (NEW)

> 플레이어마다 다른 빌드와 정체성을 만들기 위한 메타 진행 시스템.

### 13.1 시그니처 기법 (플레이어 패시브)

- 특정 재료 조합 패턴을 **N회 이상 반복 제작**하면 시그니처 기법 해금
- 해금 시 LLM이 기법 이름과 플레이버 텍스트 생성 (예: `○○가문 단야술`)
- 해금된 시그니처는 이후 동일 패턴 제작 시 자동 보정 효과 부여

#### 패턴 정의

- "패턴"의 키: **주재료 카테고리 + 속성** 조합 (예: `금속/불`, `목재/바람`)
- 해금 조건: 같은 키로 누적 10회 제작
- 한 플레이어가 보유할 수 있는 시그니처: 최대 3개 (4번째 해금 시 교체 선택)

#### 시그니처 효과 (해금 시 무작위 1개)

| 효과 | 내용 |
|---|---|
| 예리도 보정 | 해당 패턴 제작 시 예리도 +10 |
| 희귀도 보정 | 해당 패턴 제작 시 희귀도 +10 |
| 스킬 강화 | 해당 패턴 무기의 스킬을 LLM이 1티어 강하게 생성 |
| 가격 보정 | 해당 패턴 무기 판매 시 시작가 +15% |

### 13.2 전설 무기 등재

플레이어가 만든 개별 무기가 다음 조건을 충족하면 **전설 무기**로 영구 등재:

| 조건 (택1) |
|---|
| 해당 무기로 중간보스 처치 |
| 해당 무기로 용사가 5회 이상 연속 생존 |
| 해당 무기로 최종보스에게 유효타 (전투 결과 LLM 판정) |

#### 전설 등재 효과

- 무기 이름이 대장간 명예의 전당(UI 패널)에 영구 표시
- 해당 무기를 만든 **재료 조합**을 다시 사용해 제작 시:
  - 시작가 +20%
  - 첫 제시 시 용사 LLM이 전설을 언급 ("그 ○○를 만든 대장장이?")
- 플레이어 평판 영구 +5

## 15. LLM 호출 지점 정리

| 호출 위치 | 입력 | 출력 |
|---|---|---|
| 무기 이름 생성 | 재료 리스트 + 각 재료 스탯 | 무기 이름 |
| 무기 스킬 생성 | 희귀도 + 예리도 + 무기 이름 + 재료 | 스킬 텍스트 (구매 확정 시 공개) |
| 용사 첫 등장 | 직업 + 스탯 | 이름, 성격 태그, 말투 샘플 |
| 협상 대화 | NPC 상태 + 거래 이력 + 호감도 + 플레이어 제시가 | 응답 + 협상 결정 |
| 전투 결과 | 용사 스탯 + 무기 스탯 + 마왕군 스탯 | 뉴스 형식 스크립트 + 결과 코드 |
| 시그니처 명명 | 패턴 키 + 효과 | 기법 이름 + 플레이버 |
| 전리품 생성 | 마왕군 종류 + 난이도 | 재료/골드 보상 |
